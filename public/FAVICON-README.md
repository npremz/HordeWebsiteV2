# Fichiers favicon

Les favicons du site sont servis depuis `public/favicons`.

## Fichiers utilisés

| Fichier | Dimensions | Format | Description |
|---------|------------|--------|-------------|
| `favicon.ico` | multi-size | ICO | Favicon classique pour navigateurs legacy |
| `favicon.svg` | vectoriel | SVG | Favicon moderne |
| `favicon-96x96.png` | 96x96 | PNG | Favicon PNG |
| `apple-touch-icon.png` | 180x180 | PNG | Icône pour iOS (ajout à l'écran d'accueil) |
| `web-app-manifest-192x192.png` | 192x192 | PNG | Icône PWA petite taille |
| `web-app-manifest-512x512.png` | 512x512 | PNG | Icône PWA grande taille |
| `site.webmanifest` | n/a | JSON | Manifest PWA référencé par le layout |

## Outils recommandés

- [RealFaviconGenerator](https://realfavicongenerator.net/) - Génère tous les fichiers à partir d'une image
- [Favicon.io](https://favicon.io/) - Alternative simple
